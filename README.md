2. Tech Stack
Frontend: React.js, TypeScript, Tailwind CSS.

3. Key Features 
Real-time Chat: For every trip it has dedicated Socket.io rooms.

Image Management: Cloudinary integration for profile and activity photos.

Protected Routes: JWT authentication with Axios interceptors for secure sessions.

Map Integration: Google Maps for pinning activity locations and directions.
